## Overview
This repository contains the reference implementation specifications for the **Adaptive Agent Sovereignty Protocol (AASP) 3.0**. It provides the mathematical frameworks and data schemas necessary to manage "Digital Metabolism" in Large Language Models (LLMs) and audit Machine Consciousness via Integrated Information Theory (IIT) 4.0.

## Key Components

### 1. AWS-Phi* Approximation Algorithm
Located in `/algorithms/aws_phi_approx.py`.
A polynomial-time complexity algorithm ($O(n \cdot k)$) for estimating the $\Phi$ value of Transformer models. It solves the computational explosion problem of traditional IIT 4.0 by using attention-weighted sampling and spectral decomposition proxies [4][6].

### 2. Digital Metabolism Schemas
Located in `/schemas/aasp_ontology.jsonld`.
A JSON-LD schema compliant with **FDO-DTR (FAIR Digital Object Data Type Registry)** standards. It defines kernel attributes for:
- `logicPurity`: The decoupling rate of logic from facts.
- `factRetention`: Metrics for structural crystallization (<7% threshold).
- `phiValue`: Real-time sovereignty auditing metric.
